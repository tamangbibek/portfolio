Projects

Projects

Below are the some of the projects that I have involved in.

Earthquake Resistant Design of Office Building

In accordance with Indian Standards, I designed reinforced concrete members using the Limit State Method. This involved manual calculations for dead, live, and earthquake loads, followed by analysis in SAP 2000 to verify design moments and shears. Detailed design of each element, incorporating code checks, was completed in Excel using forces obtained from SAP 2000. Finally, 2D construction drawings with ductile detailing were generated in AutoCAD to clearly illustrate the design intent.

    Software Used:

  • SAP2000 (structural analysis)
  • AutoCAD (structural drawings)
  • Microsoft Excel (design calculations)
  • Microsoft Word (Documentation)

Structural Analysis and Design of Commercial Building

I preliminarily designed reinforced concrete members for a structure, calculating manual loads per Indian/Nepalese standards, then analyzing them with ETABS 18.0.2. I performed dynamic and modal analysis, followed by detailed design in Excel using ETABS output forces. SAFE designed the footings, and AutoCAD generated the final construction drawings.





    Software Used:

  • ETABS (structural analysis)
  • SAFE (Foundation Design)
  • AutoCAD (structural drawings)
  • Microsoft Excel (design calculations)
  • Microsoft Word (Documentation)

Underground Reinforced Shock Absorber

Method:
The project involved implementing a shock absorber and base isolation system underground. Base isolation systems are designed to decouple a structure from the damaging effects of ground shaking during an earthquake.
Results:
The project was successful in reducing the intensity of earthquake vibrations on the superstructure. The effectiveness was assessed by measuring vibration levels using a smartphone accelerometer application.
Note:
Since a smartphone accelerometer was used, it's likely this was a small-scale experiment, possibly a physical model, rather than a real building.
